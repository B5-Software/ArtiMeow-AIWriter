# 2. 下载安装和启动

## 2.1 下载安装包
- 访问官网下载页面，下载适合你电脑的安装包。
- 如果你是开发者，也可以下载源码。

## 2.2 安装步骤
1. 双击下载的安装包，按提示安装。
2. 安装完成后，桌面会出现“ArtiMeow AI Writer”图标。

## 2.3 启动软件
- 双击桌面图标即可启动。
- 如果你用的是开发模式：
  1. 打开命令行窗口（PowerShell）
  2. 输入：
     ```powershell
     npm install
     npm start
     ```

## 2.4 第一次启动
- 软件会自动创建小说项目文件夹。
- 会弹出欢迎界面。
